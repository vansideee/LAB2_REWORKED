# Лабораторная работа №2. Создание потоков

Проект представляет собой консольное многопоточное приложение, разработанное на языке C++ с использованием Windows API. Основная цель — демонстрация навыков работы с системными потоками, их синхронизации и разделения вычислительных задач.

##  Постановка задачи

Разработать программу, выполнение которой распределено между тремя потоками: `main` (основной), `min_max` и `average`. 

### Логика работы потоков

#### 1. Поток `Main`
Управляющий поток, выполняющий координацию работы программы:
1.  **Инициализация:** Создает массив целых чисел (размерность и значения вводятся пользователем).
2.  **Запуск потоков:** Инициирует выполнение потоков `min_max` и `average` через системный вызов `CreateThread`.
3.  **Синхронизация:** Ожидает завершения работы обоих рабочих потоков, используя функцию `WaitForSingleObject` (параметр `INFINITE`).
4.  **Обработка данных:** Заменяет минимальный и максимальный элементы исходного массива на среднее арифметическое значение, вычисленное потоком `average`.
5.  **Вывод:** Отображает итоговый массив на консоль.

#### 2. Поток `Min_Max`
Поток для поиска экстремумов:
*   Находит минимальный и максимальный элементы массива.
*   Выводит результаты поиска на консоль.
*   **Имитация нагрузки:** После каждой операции сравнения элементов выполняется задержка выполнения на **7 мс** (`Sleep(7)`).

#### 3. Поток `Average`
Поток для статистических вычислений:
*   Вычисляет среднее арифметическое элементов массива.
*   Выводит результат на консоль.
*   **Имитация нагрузки:** После каждой операции суммирования элементов выполняется задержка выполнения на **12 мс** (`Sleep(12)`).

---

##  Технический стек

*   **Язык:** C++ (Standard: C++17)
*   **API:** Windows API (Native threads)
*   **IDE:** Visual Studio 2022
*   **Ключевые используемые функции:**
    *   `CreateThread` — для порождения параллельных потоков.
    *   `WaitForSingleObject` — для синхронизации главного потока с рабочими.
    *   `Sleep` — для реализации задержек (имитация длительных вычислений).
